Fill in these values first:
- Building name: [BUILDING_NAME]
- Internal kind key: [camelCaseKind]
- Folder name: [snake_or_lowercase_folder_name]
- Icon: [EMOJI]
- Map symbol: [1-letter symbol]
- Theme direction: [wood / stone / metal / farm / food / magic / etc.]
- Cost: [example: log 50, stone 50]
- Destroy refund: [example: log 25, stone 25]
- Requirements to build: [example: horseWagon x1, storage x1]
- Build difficulty: [example: 3.4]
- Footprint width: [example: 2]
- Footprint height: [example: 2]
- Rotatable: [true/false]
- Sprite file name: [example: building_my_new_building_64x64.svg]

	`src/buildings/[folder_name]/[folder_name]_ui.js`
